# React starter

[https://najwer23.github.io/react-starter/](https://najwer23.github.io/react-starter/)

## Installation

```sh
$ yarn install
```

## Development

```sh
$ yarn start
```

## Production

```sh
$ yarn build
```

## Deploy to gh-pages

```sh
$ yarn deploy
```

## Tools

#### Husky

```sh
yarn husky install
```

#### Sort package.json

```sh
npx sort-package-json
```

#### Self Update

```sh
ncu -u
```

#### Yarn update

```sh
yarn set version latest
```
