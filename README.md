
# React starter
[https://najwer23.github.io/react-starter/](https://najwer23.github.io/react-starter/)

## Self Update
```sh
ncu -u
```

## Yarn update
```sh
yarn set version latest
```

## Installation
```sh
$ yarn install
```

## Development
```sh
$ yarn start
```

## Production
```sh
$ yarn build
```

## Deploy to gh-pages
```sh
$ yarn deploy
```

### Husky
```sh
yarn husky install
```
