# React Starter

**Website:** https://najwer23.github.io/react-starter

### Self Update

```
ncu -u
```

### Yarn Update

```
yarn set version latest
```

### Installation

```sh
yarn install
```

### Development

```sh
yarn start
```

### Production

```sh
yarn build
```

### Deploy to gh-pages

```sh
yarn deploy
```
